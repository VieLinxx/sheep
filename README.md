# 校园抓捕游戏

一个基于Three.js的3D校园抓捕游戏，玩家可以扮演羊或狼进行追逐游戏。

## 游戏说明

### 游戏规则
1. **躲藏阶段**：游戏开始时有30秒的躲藏时间，此时玩家无法移动
2. **追逐阶段**：躲藏时间结束后，游戏正式开始，持续3分钟
3. **胜利条件**：
   - 狼：在时间内抓住所有羊
   - 羊：在时间内成功逃脱

### 角色说明
- **羊**：白色绒球帽标识，目标是躲避狼的追捕
- **狼**：尖耳朵标识，可以使用钩子抓住羊

### 操作方式

#### 移动控制
- **左摇杆**：控制角色移动方向
- **键盘WASD/方向键**：上/下楼梯
- **数字键1/2/3**：快速切换楼层
- **空格键**：从天台跳下（在高层时）

#### 钩子功能（仅狼可用）
- **右摇杆**：瞄准并发射钩子
  - 拖动摇杆瞄准目标
  - 松开摇杆发射钩子
- 钩子命中羊后，羊会被捆绑到最近的椅子上

### 游戏场景
- **教学楼**：两层建筑，有天台可以跳下
- **图书馆**：单层建筑
- **宿舍楼**：单层建筑
- **食堂**：单层建筑
- **操场设施**：篮球场、滑梯、秋千
- **装饰物**：树木、花坛、长椅、路灯

### 特色功能
1. **多层建筑系统**：可以通过楼梯在不同楼层间移动
2. **钩子系统**：狼可以使用钩子远程抓住羊
3. **碰撞检测**：玩家无法穿过建筑物
4. **音效系统**：发射钩子、抓住羊等都有音效反馈
5. **视觉效果**：钩子飞行动画、瞄准线显示

## 技术栈
- **Three.js**：3D渲染引擎
- **nipplejs**：虚拟摇杆控制
- **原生JavaScript**：游戏逻辑

## 如何开始
1. 直接在浏览器中打开 `index.html` 文件
2. 等待30秒躲藏时间结束
3. 使用左摇杆移动角色
4. 如果是狼，使用右摇杆发射钩子抓住羊

## 注意事项
- 游戏需要在支持WebGL的现代浏览器中运行
- 移动设备上可以使用触摸屏控制
- 建议使用Chrome、Firefox或Edge浏览器
