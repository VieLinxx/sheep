# 校园大逃亡（捉迷藏版）- 开发进度报告

## 已完成功能

### 1. 核心游戏机制 ✅
- **阵营系统**：实现了羊（蓝队）和狼（红队）的阵营分配
- **角色颜色**：羊为皇家蓝，狼为深红色，便于区分
- **生命系统**：玩家有2条命，第一次被抓变狼，第二次才淘汰
- **变身机制**：角色被抓后可以变身，颜色和帽子会相应改变

### 2. 角色外观系统 ✅
- **羊角帽**：弯曲的羊角 + 粉色耳朵，符合小羊形象
- **狼耳帽**：尖三角形耳朵 + 红色眼睛 + 黑色鼻子，符合狼人形象
- **动态切换**：角色身份改变时，帽子和颜色自动更新

### 3. 相机系统 ✅
- **固定视角**：相机始终在角色后方偏上位置（距离7米，高度3.5米）
- **平滑跟随**：使用lerp插值，相机移动平滑不抖动
- **FOV优化**：设置为65度，减少眩晕感
- **不跟随旋转**：相机不随鼠标/手指旋转，始终保持固定角度

### 4. 狼人技能系统 ✅
- **钩子技能**：
  - 右摇杆瞄准 + 松手发射
  - 20秒冷却时间
  - 命中后将羊拉到面前
  - 视觉效果：钩子飞行动画 + 瞄准线
  
- **加速技能**：
  - 点击右侧⚡按钮激活
  - 速度提升1.8倍，持续5秒
  - 20秒冷却时间
  - UI显示冷却倒计时

### 5. 小羊能力 ✅
- **解救队友**：
  - 靠近被捆绑的队友时显示解救按钮
  - 长按10秒完成解救
  - 中途松开进度归零，可继续解救
  - 可被狼人钩子打断
  
- **变身机制**：
  - 第一次被抓：被捆绑等待解救
  - 第二次被抓：直接感染成新狼人

### 6. 道具系统 ✅
- **加速鞋** 🏃：移动速度×1.8，持续5秒
- **复活护符** ❤️：增加1条命（第三次被抓才变狼）
- **隐身斗篷** 👻：身体变半透明，狼人无法瞄准，持续5秒
- **净化工具** ✨：可解救被感染的队友恢复成羊
- **刷新机制**：每30秒刷新，最多同时存在6个

### 7. 游戏流程 ✅
- **躲藏期**：30秒，小羊可移动，狼人无法移动（观望动画）
- **追逐期**：20分钟，双方正常游戏
- **胜利条件**：
  - 狼人：20分钟内感染所有小羊
  - 小羊：时间到时人数更多

### 8. UI界面 ✅
- **房间大厅**：输入昵称、创建/加入房间
- **等待房间**：显示房间码和玩家列表
- **游戏UI**：
  - 计时器显示
  - 角色身份标识（羊/狼）
  - 生命值显示（❤️❤️）
  - 技能按钮（狼人专用）
  - 解救按钮（小羊专用）
  - 道具提示通知
  - 门交互按钮

### 9. 其他功能 ✅
- **音效系统**：钩子发射、抓住、道具拾取等音效
- **视觉效果**：隐身半透明、加速特效等
- **碰撞检测**：玩家无法穿过建筑物
- **楼层系统**：可通过楼梯上下楼，可从天台跳下

## 技术架构

### 文件结构
```
sheep play/
├── index.html          # 游戏主页面（包含所有UI界面）
├── style.css           # 样式文件（响应式设计）
├── js/
│   └── game.js         # 游戏主逻辑（约2200行）
└── README.md           # 说明文档
```

### 核心类：Game
主要属性：
- `playerRole`: 玩家角色（'sheep' 或 'wolf'）
- `playerLives`: 玩家生命值
- `gameState`: 游戏状态（'lobby', 'hiding', 'playing', 'ended'）
- `hookSystem`: 钩子系统
- `speedBoost`: 加速技能
- `rescueSystem`: 解救系统
- `activeItems`: 激活的道具
- `cameraConfig`: 相机配置

主要方法：
- `createPlayer()`: 创建角色
- `createRoleHats()`: 创建角色帽子
- `setPlayerRole()`: 切换角色身份
- `updateCamera()`: 更新相机位置
- `activateSpeedBoost()`: 激活加速技能
- `pickupItem()`: 拾取道具
- `animate()`: 游戏主循环
